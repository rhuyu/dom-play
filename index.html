Welcome to a script featuring a scene from Rush Hour 3!

Here we have a demonstration of how we can highlight only certain actor's lines by implementing javascript. This is done by assigning each actor's lines with a data attribute and creating a function that triggers an event to highlight certain sets of lines while changing the other actor's lines back to normal!

Click on any of the actor's lines and watch as the corrolating lines highlight!
